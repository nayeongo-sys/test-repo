# 공지사항 게시판

React + TypeScript + Vite로 구현된 공지사항 게시판입니다.

<br />

https://github.com/user-attachments/assets/e21a37db-d98c-4f85-89cd-50f49aca8c9b




<br />

## 기능

- 공지사항 목록 조회 및 검색
- 공지사항 작성 (CKEditor, Tiptap 에디터 지원)
- 공지사항 수정
- 공개/비공개 설정

<br />

## 에디터 라이브러리 비교

| 라이브러리 | 장점 (요구 기능 관점) | 단점 / 고려할 점 |
| --- | --- | --- |
| **TipTap** | - `@tiptap/extension-text-align`으로 정렬 지원 (left, center, right, justify)<br>- 폰트 스타일, 색상, 형광펜 등 다양한 커스텀 extension 구축 가능<br>- 이미지, 테이블, 링크 등 삽입 확장성 매우 높음<br>- ProseMirror 기반이라 구조적이고 안정적<br>- Next.js + React 환경에서 훌륭한 지원<br>- **HTML, JSON, Markdown** 출력 지원 | - 초기 설정이 조금 복잡할 수 있음<br>- UI 컴포넌트(툴바 등)를 직접 만들어야 할 수 있음<br>- 번들 사이즈 증가 가능성 및 커스터마이징 유지보수 비용 고려 필요 |
| **CKEditor (CKEditor 5)** | - 매우 성숙하고 안정된 WYSIWYG<br>- 정렬(문단 정렬), 들여쓰기, 서체 크기/굵기/색상, 밑줄, 하이라이트(형광펜) 플러그인 존재<br>- 이미지 삽입, 링크, 표 삽입 등 고급 편집 기능 풍부<br>- React용 래퍼 제공, 확장 플러그인 생태계 매우 크고 문서도 많음 | - 빌드 크기 클 수 있음<br>- 커스터마이징이 복잡할 수 있고, 라이센스 고려 필요 (상업용에서는 비용 고려)<br>- Next.js SSR 고려 시 초기 로딩 전략 필요 |
| **Quill / React-Quill** | - 기본적인 텍스트 스타일링(굵기, 이탤릭, 밑줄), 리스트, 링크, 이미지 삽입 기본 지원<br>- 비교적 경량 / 사용이 쉬움<br>- React에 친화적<br>- 빠르게 세팅 가능 | - 정렬 기능은 제공하지만, 들여쓰기나 좀 더 복잡한 스타일은 제한적일 수 있음<br>- 테이블 삽입 기능이 기본 제공이 약함<br>- 매우 복잡한 커스텀 구조에는 한계가 있음 |
| **Slate.js** | - 가장 자유도가 높음 (리치 텍스트 + 구조적 문단 + 커스텀 블록 완벽 제어 가능)<br>- 정렬, 들여쓰기, 서체 스타일, 테이블, 링크 모두 직접 확장 가능<br>- React와 매우 잘 맞음 | - 구축 초기 비용이 매우 큼<br>- 기본 UI 제공이 거의 없어서 툴바 등 UI를 직접 만들어야 함<br>- 유지보수 어려움 + 버그 위험성 있음 |
| **Lexical (Meta)** | - React 친화적이고 최신 설계<br>- 정렬, 스타일, 삽입 등 다양한 기능 확장 가능<br>- 비교적 경량 + 성능 고려 설계됨<br>- 점점 인기가 높아지고 있음 | - 아직 일부 고급 플러그인은 직접 구현해야 할 수 있음<br>- 문서나 예제가 TipTap / CKEditor만큼 풍부하지 않을 수 있음<br>- 커스터마이징에는 학습 곡선이 있을 수 있음 |

## 실행 방법

```bash
# 의존성 설치
pnpm install

# 개발 서버 실행
pnpm dev
```

브라우저에서 `http://localhost:5173`으로 접속하여 확인할 수 있습니다.
